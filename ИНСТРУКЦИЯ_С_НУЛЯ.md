# Игровой хаб — инструкция с нуля

Пошагово: что установить, что нажать, что ввести. Делай по порядку.

---

## Шаг 0. Где у тебя проект

Папка с игрой у тебя здесь:
`C:\Users\axilb\OneDrive\Рабочий стол\PROGS\GAME_HUB`

Внутри должны быть папки: `app`, `bot`, `server` и файлы `package.json`, `SPEC.md` и т.д.

## Шаг 1. Установить Node.js

Node.js — это программа, на которой работает и сервер, и приложение.

1. Открой в браузере: **https://nodejs.org**
2. Скачай версию с надписью **LTS** (большая зелёная кнопка).
3. Запусти установщик и нажимай «Next», ничего не снимая с галочек. В конце нажми «Finish».
4. **Перезапусти компьютер** (или закрой все окна и открой заново).
5. Проверь: нажми на клавиатуре **Win + R**, впиши `cmd` и нажми Enter. В чёрном окне введи:
   ```text
   node -v
   ```
   и нажми Enter. Должно появиться что-то вроде `v20.10.0`. Если видишь число с буквой v — всё ок.

## Шаг 2. Открыть проект в терминале

1. Нажми **Win + R**, впиши `cmd`, Enter — откроется чёрное окно (командная строка).
2. Перейди на рабочий стол и в папку с проектом. Вводи команды по одной и каждый раз нажимай Enter:

   ```text
   cd Desktop
   cd PROGS
   cd GAME_HUB
   ```

   Если у тебя путь с «Рабочий стол», попробуй так:

   ```text
   cd OneDrive
   cd "Рабочий стол"
   cd PROGS
   cd GAME_HUB
   ```

   Внизу слева должно быть что-то вроде `...\GAME_HUB>` — значит ты в нужной папке.

## Шаг 3. Установить зависимости проекта

В той же командной строке (в папке GAME_HUB) вводи по очереди, после каждой нажимай Enter:

```text
npm install
```

Подожди, пока всё скачается (может быть 1–2 минуты). Ошибок быть не должно.

Потом по очереди:

```text
cd server
npm install
cd ..
```

```text
cd app
npm install
cd ..
```

```text
cd bot
npm install
cd ..
```

После этого ты снова должен быть в папке GAME_HUB (видно по строке `...\GAME_HUB>`).

## Шаг 4. Создать файл .env

1. Открой папку проекта в проводнике:
   `C:\Users\axilb\OneDrive\Рабочий стол\PROGS\GAME_HUB`
2. Найди файл **`.env.example`** (может называться просто `env.example` — без точки в начале).
3. Скопируй этот файл (правой кнопкой → Копировать, потом Вставить) и переименуй копию в **`.env`** (точка в начале, без «example»).
4. Открой **`.env`** блокнотом (правой кнопкой → Открыть с помощью → Блокнот).
5. Найди строку `BOT_TOKEN=` и после знака `=` вставь свой токен бота (тот, что от BotFather). Должно получиться так:
   ```text
   BOT_TOKEN=8265553142:AAHaX9RpN4_84usEpmioqRdwH7BF1oW5SYc
   ```
6. Остальные строки можно не трогать. Сохрани файл (Файл → Сохранить) и закрой блокнот.

## Шаг 5. Запустить игру

1. Снова открой командную строку и перейди в папку проекта (как в шаге 2):
   ```text
   cd OneDrive
   cd "Рабочий стол"
   cd PROGS
   cd GAME_HUB
   ```
   (или твой вариант пути до GAME_HUB.)

2. Введи одну команду и нажми Enter:
   ```text
   npm run dev
   ```

3. Появится куча текста — это нормально. Должны быть строки вроде:
   - что-то про порт 3000 (сервер),
   - что-то про порт 5173 (приложение),
   - «Bot running» (бот).

4. **Окно не закрывай** — пока оно открыто, игра работает.

---

## Шаг 6. Открыть игру в браузере

1. Открой браузер (Chrome, Edge или любой).
2. В адресной строке введи:
   ```text
   http://localhost:5173
   ```
   и нажми Enter.

3. Должна открыться страница «Игровой хаб» с кнопками «Создать комнату» и полем «Войти по коду». Можешь нажать «Создать комнату» — появится лобби с кодом и списком игроков.

Если страница не открылась — проверь, что в шаге 5 окно с `npm run dev` не закрыто и в нём нет красных ошибок.

---

## Шаг 7. Проверить бота в Telegram

1. Открой Telegram (на телефоне или в программе).
2. Найди бота **@tg_gamehub_bot** (поиск по имени).
3. Нажми «Запустить» или напиши `/start`.
4. Должна появиться кнопка **«Играть»**. По ней откроется приложение. Если откроется пусто или с ошибкой — это из-за того, что Telegram хочет открывать приложение по ссылке с интернета (по IP или домену). Локально в браузере (шаг 6) всё уже работает.

---

## Если что-то пошло не так

- **«node не является внутренней или внешней командой»** — Node.js не установился или не виден. Поставь заново с nodejs.org и перезагрузи компьютер, потом снова открой cmd.
- **«npm не является внутренней или внешней командой»** — то же самое: переустанови Node.js и перезагрузись.
- **Ошибка при `npm install`** — пришли сюда текст ошибки (можно скопировать из чёрного окна).
- **Страница localhost:5173 не открывается** — убедись, что в окне с `npm run dev` нет красных сообщений и что ты не закрывал это окно.

## Остановить игру

В окне, где запущено `npm run dev`, нажми **Ctrl + C** (зажать Ctrl и нажать C). Спросят «Terminate batch job?» — введи букву **Y** и Enter. Окно можно закрыть.

Когда захочешь снова запустить — снова открой cmd, зайди в папку GAME_HUB и введи `npm run dev`.
