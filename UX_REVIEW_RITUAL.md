# UX Review Ritual

Weekly ritual to keep GameHub polished and compact.

## 1) Input
- Pick 10 recent user sessions (or internal playtests).
- Collect top friction points from Home -> Lobby -> Match flow.

## 2) Review (30 minutes)
- Rank top-20 pains by impact (blocked / annoying / cosmetic).
- Mark each pain with owner and target fix window.

## 3) Action
- Fix top 3 blocked pains immediately.
- Add tests or smoke checks for each fixed pain.

## 4) Track
- Compare conversion and completion KPIs week-to-week.
- Update the "top-20 pains" list and close resolved items.
