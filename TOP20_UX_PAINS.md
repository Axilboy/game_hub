# Top-20 UX Pains (Current Snapshot)

1. Invite link expired message appears too late for some users.
2. Host settings in lobby can feel overloaded for first-time hosts.
3. Game selection is not always obvious on smaller screens.
4. Some game result screens lack context for newcomers.
5. Retry guidance on network timeout could be clearer.
6. User may not notice reconnect banner quickly enough.
7. Dense statistics blocks can overwhelm casual players.
8. Shop cards do not yet differentiate "value now" vs "future".
9. Truth/Dare category intent is not always obvious from names.
10. Bunker phase understanding requires initial explanation.
11. Multiple modals can stack cognitive load in one session.
12. Keyboard-only navigation is still underused by users.
13. Error copy is improved but not fully unified in all branches.
14. Rounds with many players can feel visually busy.
15. Player role hints may be missed due to compact card layout.
16. Last-room and rematch CTAs compete for attention.
17. No explicit "what changed" hints after settings presets.
18. Some advanced settings should auto-collapse after start.
19. Restore purchase flow is local-only, not cloud-backed yet.
20. New game onboarding still depends on host discipline.
